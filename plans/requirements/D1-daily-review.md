# D1 — 每日复盘

> 需求总览索引：[requirements.md](requirements.md)

---

## 1. [用户故事]()

每天收盘后，从各数据源汇总当天信息，生成一份格式标准的复盘文档。
用来训练盘感、为次日交易提供思路。
复盘内容持久化保存，后续可以随时翻看历史复盘。

**用户原话：**

> "需要一个复盘标准，从各个数据源汇总，形成一段格式标准复盘文档，可以给我看，去训练盘感，给明天的交易提供思路，也需要持久化，后续可以翻看前几天的复盘内容"

---

## 2. 功能拆解

### 2.1 复盘文档模板

复盘文档有固定的格式标准，包含以下章节：

| 章节     | 内容                         | 数据来源               |
| -------- | ---------------------------- | ---------------------- |
| 大盘表现 | 主要指数涨跌幅、成交额       | 日线行情               |
| 热点板块 | 当日涨幅/跌幅前 5 板块       | 待定（后续补充数据源） |
| 资金动向 | 北向资金净买卖、主力流入流出 | 资金流向、北向资金     |
| 龙虎榜   | 上榜个股、净买入额、上榜原因 | 龙虎榜                 |
| 重要政策 | 当日相关政策关键词筛选结果   | 政策抓取               |
| 市场情绪 | 论坛热度 / 情绪指数          | 情绪采集、手动录入     |
| 个人笔记 | 用户自己的观察和次日思路     | 用户输入               |

### 2.2 复盘文档生成

- 支持一键生成当天复盘（拉取最新数据 → 填入模板 → 生成文档）
- 如果部分数据源拉取失败，自动跳过该章节并标记，不影响其他章节
- 生成后自动持久化存储

### 2.3 复盘查看

- 查看当天复盘
- 按日期翻查历史复盘
- 列表模式：日期 + 关键摘要（大盘涨跌、北向资金净额等）

### 2.4 个人笔记

- 每篇复盘文档支持追加个人笔记
- 笔记独立保存，不覆盖自动生成的内容
- 翻看历史复盘时可同时看到当时写的笔记

---

## 3. 已确认的决定

- **文档形式：** Markdown 格式，可直接阅读，也支持一键转为带格式的文档
- **导出：** 支持导出（Markdown 文件 / 带格式文档）
- **个人笔记：** 直接在复盘文档上追加输入，笔记不覆盖自动生成内容
- **热点板块数据源：** 待定，后续讨论
