## 1. 目标

从技术面角度辅助判断趋势方向、识别主力意图、标记机会与风险信号，为复盘和次日交易提供参考。

---

## 2. 核心思路

### 2.1 三重滤网（看大做小）

> 引用 Alexander Elder 的三重滤网系统，结合 A 股习惯调整。

| 层级                 | 周期               | 工具           | 目的                       |
| -------------------- | ------------------ | -------------- | -------------------------- |
| **第一重（大趋势）** | 周线（或 60 分钟） | MA20/MA60      | 定方向：趋势是多还是空     |
| **第二重（中周期）** | 日线               | MACD / K线形态 | 找拐点：等回调到支撑再入场 |
| **第三重（小周期）** | 30 分 / 15 分      | 价格行为       | 找时机：等小周期确认信号   |

**看大做小规则：**

| 第一重（大趋势）     | 第二重（中周期）              | 允许操作       |
| -------------------- | ----------------------------- | -------------- |
| 多头（价格 > MA20↑） | 回调到 MA60/支撑区 + K 线止跌 | ✅ 逢低做多    |
| 空头（价格 < MA20↓） | 反弹到 MA60/压力区 + K 线滞涨 | ✅ 逢高做空    |
| 震荡（MA20 走平）    | 价格在区间内                  | ⏸ 观望，等突破 |

### 2.2 价格行为 — 揣摩主力意图

> 不靠指标，靠裸 K 线 + 关键位置分析。

| 信号         | 看什么                                                         | 含义                       |
| ------------ | -------------------------------------------------------------- | -------------------------- |
| **突破确认** | 放量突破关键位置（前高 / 颈线 / MA60），且收盘站稳             | 主力真突破，跟进           |
| **假突破**   | 突破后下一根 K 线立刻缩量回调，回到区间内                      | 主力诱多/诱空，反向操作    |
| **底分型**   | 日线三根 K 线：低点更低 → 低点未破新低 + 收盘过半 → 高点更高   | 潜在止跌反转               |
| **顶分型**   | 日线三根 K 线：高点更高 → 高点未创新高 + 收盘破前低 → 低点更低 | 潜在见顶回落               |
| **长上影**   | 日内冲高回落，收盘在下 1/3 区域，放量                          | 主力出货 / 上方抛压大      |
| **长下影**   | 日内探底回升，收盘在上 1/3 区域，放量                          | 主力吸筹 / 下方支撑强      |
| **缺口**     | 跳空高开 / 低开不回补                                          | 强势信号，不回补则趋势延续 |

> 价格行为分析的**核心原则：** 关键位置 + K 线形态 + 放量确认，三者缺一可信度都降低。

### 2.3 量价关系 — 机会与风险标识

| 量价组合        | 位置            | 判定                | 操作建议           |
| --------------- | --------------- | ------------------- | ------------------ |
| **放量上涨** ☀️ | 突破 / 上涨初期 | ✅ 正常，趋势健康   | 持有 / 顺势加仓    |
| **缩量上涨** 🌤 | 上涨一段后      | ⚠️ 背离，动能减弱   | 减仓保护利润       |
| **放量滞涨** ⛈  | 高位 / 压力区   | 🔴 危险，主力出货   | 清仓离场           |
| **缩量下跌** 🌧 | 下跌初期        | ✅ 正常，无人接盘   | 观望，不抄底       |
| **放量下跌** ⛈  | 破位 / 高位     | 🔴 危险，恐慌出逃   | 清仓离场           |
| **缩量止跌** 🌤 | 下跌一段后横盘  | ⚠️ 可能到底，需确认 | 关注，等待放量反转 |
| **放量反转** ☀️ | 底部 / 支撑区   | ✅ 主力进场，机会   | 试探性建仓         |

> **关键判断逻辑：** 同样的量价组合，出现在不同位置意义完全不同。
> 例如"放量" = 高位是风险，低位是机会。

## 3. 功能拆解

### 3.1 均线计算

- 支持 MA（简单移动均线）：MA5、MA10、MA20、MA60
- 判断均线状态：多头排列（短 > 长 ↑）、空头排列（长 > 短 ↓）、粘合（缠绕走平）
- 判断价格相对均线的位置：上方 / 下方 / 穿越

### 3.2 三重滤网信号

输入：某只股票 + 当前日期
输出：三重滤网结论（多头 / 空头 / 震荡 / 观望）

```
示例输出：
第一重（周线）：MA20 > MA60，趋势 ↑ → 多头
第二重（日线）：价格回调至 MA60，K 线收长下影 → 支撑有效
第三重（30分）：放量突破短期下降趋势线 → 确认
结论：✅ 多头机会，可逢低入场（三级确认）
```

### 3.3 价格行为识别

- 识别常用 K 线组合：顶点分型、底分型、长上影、长下影、缺口
- 结合关键位置（前期高低点、均线）标注信号

### 3.4 量价关系标注

- 每日收盘后，对股票池中每只股票标注量价组合标签
- 标注当前的位置（突破 / 高位 / 底部 / 下跌途中）
- 输出风险/机会标识（☀️ / ⚠️ / 🔴）

## 5. 数据依赖

| 需要的数据             | 来源             | 频次     |
| ---------------------- | ---------------- | -------- |
| 日线行情（开高低收量） | 已有 D1 数据采集 | 每天     |
| 周线行情（由日线聚合） | 日线按周聚合计算 | 每天计算 |

> 技术面分析所需的数据**全部来自日线行情**，不依赖外部新数据源，实现成本低。

## 6. 已确认的决定

- **三重滤网周期参数：** 可调（用户可配置各层级的均线周期和指标）
- **分析范围：** 全股票池（不限于持仓/关注）
- **量价连续过滤：** 需要，连续 N 天同一信号才算有效（N 默认 2，可调）
