# 情绪周期分析 — 计划

> 本文档属于 `plans/analysis/` 目录，分析与策略相关 plan 放这里。

---

## 1. 目标

从市场情绪角度判断当前"水温"、识别热点板块、评估板块延续性和赚钱效应，辅助回答三个问题：

1. **现在能不能做？** — 水温判断（市场整体环境）
2. **做什么方向？** — 热点识别（主线板块）
3. **能持续多久？** — 延续性 + 赚钱效应

---

## 2. 功能拆解

### 2.1 水温判断（市场情绪温度计）

#### 核心指标

| 指标             | 说明                                                | 数据来源     |
| ---------------- | --------------------------------------------------- | ------------ |
| **成交额**       | 全市场 / 沪深两市合计，计算相对于 20 日均线的偏离度 | 日线行情汇总 |
| **涨停家数**     | 当日涨停股数量（剔除 ST）                           | 日线行情统计 |
| **跌停家数**     | 当日跌停股数量（剔除 ST）                           | 日线行情统计 |
| **连板高度**     | 当日最高连板数（如 5 连板）                         | 日线行情统计 |
| **涨跌家数比**   | 上涨家数 / 下跌家数                                 | 日线行情统计 |
| **昨日涨停溢价** | 昨日涨停股今天平均涨幅                              | 日线行情计算 |

#### 温度区间

| 温度        | 成交额特征            | 涨停   | 连板高度 | 对应操作                   |
| ----------- | --------------------- | ------ | -------- | -------------------------- |
| **冰点** ❄️ | 低于 MA20 的 30% 以上 | < 20   | ≤ 2      | 空仓 / 轻仓试错            |
| **低温** 🌥 | 低于 MA20 的 10%-30%  | 20-40  | 3-4      | 谨慎参与，小仓试探         |
| **常温** ☀️ | 在 MA20 上下 10% 以内 | 40-70  | 4-6      | 正常参与，主线仓位可重     |
| **高温** 🔥 | 高于 MA20 的 10%-30%  | 70-100 | ≥ 6      | 积极参与，但警惕高潮转退潮 |
| **沸点** 🌋 | 高于 MA20 的 30% 以上 | > 100  | 极高     | 高潮区，逐步减仓防退潮     |

> 成交额偏离度计算公式：(当日成交额 - MA20) / MA20 × 100%。
> 例如：MA20 = 8000 亿，当日 9500 亿 → 偏离度 +18.75% → 高温区间。

### 2.2 热点识别（主线板块）

#### 板块排行

| 维度         | 看什么                              | 识别方式          |
| ------------ | ----------------------------------- | ----------------- |
| **行业涨幅** | 申万一级行业 / 概念板块当日涨幅排名 | ak share 板块行情 |
| **涨停密度** | 板块内涨停家数 / 板块总家数         | 统计              |
| **资金流入** | 板块主力资金净流入排名              | 资金流向数据      |
| **龙头股**   | 板块内涨幅最大 / 最先涨停的个股     | 日线行情          |

#### 主线判定

| 条件                                         | 判定                  |
| -------------------------------------------- | --------------------- |
| 板块涨幅排名前 3 + 涨停家数 > 5 + 资金净流入 | ✅ 主线热点           |
| 板块涨幅靠前但涨停家数 < 3                   | ⚠️ 跟风轮动，非主线   |
| 板块连续 3 天进入前 5                        | ✅ 主线确认，延续性强 |
| 前一天主线板块今日跌出前 10                  | ⚠️ 主线可能切换       |

### 2.3 板块延续性评估

| 维度         | 指标                                     | 强势信号            |
| ------------ | ---------------------------------------- | ------------------- |
| **持续性**   | 板块排名连续在前 5 的天数                | ≥ 3 天 → 主线行情   |
| **内部结构** | 板块内是否有连板龙头                     | 有龙头 → 行情有高度 |
| **梯队**     | 板块内涨停股是否分梯队（龙头→跟风→补涨） | 梯队完整 → 延续性强 |
| **成交量**   | 板块成交量是否持续放大                   | 量增价涨 → 趋势健康 |

#### 板块生命周期

```
启动期（1-2 天）→ 主升期（3-6 天）→ 高潮期（1-2 天）→ 退潮期
 个别股涨停    板块联动，梯队完整   满屏涨停，跟风扩散    龙头倒下，批量炸板
```

### 2.4 连板梯队分析

> 连板梯队是情绪周期的"骨架"——梯队在，行情在；梯队塌了，行情结束。

#### 梯队结构

```
高度板（最高连板）
  ├── 7板以上  → 妖股周期，情绪极亢奋
  ├── 5-6板    → 主线龙头，行情主升段
  ├── 3-4板    → 中位股，板块效应扩散中
  ├── 2板      → 跟风股，确认板块强度
  └── 首板     → 基础量，决定板块宽度
```

#### 关键指标

| 指标           | 含义                   | 健康信号                  | 危险信号                                |
| -------------- | ---------------------- | ------------------------- | --------------------------------------- |
| **最高连板**   | 市场高度天花板         | ≥ 5 板，且未断板          | 最高板炸板 / 跌停                       |
| **梯队完整度** | 各层级是否都有股在     | 首板→2板→3板→高度板均有人 | 中间断层（如只有5板+首板，没有2/3/4板） |
| **晋级率**     | 低位晋级高位的比例     | 首板→2板 > 30%            | < 10%，接力意愿极差                     |
| **断板股表现** | 前日连板股今天是否跌停 | 断板后温和调整            | 断板即跌停 → 亏钱效应炸裂               |
| **板块分布**   | 连板股集中在哪个板块   | 集中在一个板块 → 主线清晰 | 分散在多个板块 → 没有合力               |

#### 梯队分析示例

```
📊 今日连板梯队（2026-06-15）

高度板：  5板  XXXXX（AI算力）
         4板  XXXXX（AI算力）
中位股：  3板  2只（AI算力1、机器人1）
         2板  5只（AI算力2、机器人2、消费电子1）
首板：    42只（AI算力15、机器人8、消费电子5、其他14）

结论：
✅ 梯队完整，主线清晰（AI算力）
✅ 晋级率 35%，接力意愿强
⚠️ 高度板 5板 未突破 6板，关注明天能否晋级
```

#### 梯队风险信号

| 场景                    | 判定        | 应对                 |
| ----------------------- | ----------- | -------------------- |
| 高度板断板 + 中位股全倒 | 🔴 退潮确认 | 空仓                 |
| 高度板还在但中位股断层  | ⚠️ 分歧日   | 等次日确认方向       |
| 高度板 + 梯队完整       | ✅ 主升延续 | 继续参与             |
| 没有高度板但首板很多    | ⚠️ 混沌期   | 轻仓试错，等待新方向 |

### 2.5 赚钱效应评估

| 指标             | 计算方法                   | 赚钱效应                 |
| ---------------- | -------------------------- | ------------------------ |
| **昨日涨停溢价** | 昨日涨停股今日平均涨幅     | > 2% → 好；< -1% → 差    |
| **炸板率**       | (炸板数 / 涨停总数) × 100% | < 20% → 好；> 40% → 差   |
| **连板晋级率**   | 今日 2 连板 / 昨日首板     | > 30% → 接力意愿强       |
| **涨跌家数比**   | 上涨 / 下跌                | > 2 → 普涨；< 0.5 → 普跌 |

## 5. 已确认的决定

- **温度区间阈值：** 可调（用户可自定义各温度区间的偏离度百分比）
- **板块分类：** 申万一级行业和概念板块**同时看**，分析时标注来源
- **计算时机：** 触发复盘时计算，不预跑
